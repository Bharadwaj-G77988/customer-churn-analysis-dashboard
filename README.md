# Customer Churn Analysis Dashboard

## Overview
This project analyzes customer churn behavior using data analysis and visualization techniques to identify patterns influencing customer retention.

## Tools Used
- Python (Pandas, NumPy)
- Power BI
- SQL (concepts)
- Data Cleaning & Exploratory Data Analysis

## Key Features
- KPI Metrics (Churn Rate, Average Tenure, Monthly Charges, Total Customers)
- Interactive Filters (Contract, Gender, Payment Method)
- Customer Segmentation Analysis
- Business Insights for Retention Strategies

## Files Included
- Power BI Dashboard (.pbix)
- Dataset
